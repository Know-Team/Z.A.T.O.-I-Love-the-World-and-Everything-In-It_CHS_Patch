# 识渡汉化组Z.A.T.O.汉化补丁/°Know Team Z.A.T.O. I Love the World and Everything In It_CHS_Patch

适用系统：Windows

运行方式：似乎得先解包.rpa再覆盖相关文件

简中字体确定为：IBMPlexSansSC-Text

字体理念：Courier New Font（即英文原版选用字体）是用于IBM 打字机，所以颇有一种打字机的感觉。它让我想到好莱坞剧本（结果一查，就是好莱坞剧本的推荐字体）中文其实很难找到相对应的字体，基本上追求铅印或油墨感，而丢失了那种打字机钢笔的感觉（？）。我一时间不知道该如何把握，所以我选择了有渊源的IBMPlex，而非Cascadia Next或Plix普力士，因为差别不算太大。而且字重方面为了增加可读性，选用了Text，而非Light或ExtraLight,与其形似，可读性才是第一位的。坏消息就是，IBMPLEX-TEXT看起来太过普通了一些。
